Simple RSS feed viewer on CUI
=============================

A simple RSS feed viewer for CUI.

Just preview the RSS feed.



## Installation
### Depends
- Ruby 3.*


### How to install
```sh
$ curl -L https://raw.githubusercontent.com/mitsu-ksgr/rss-viewer/refs/heads/main/rssv.rb -o ./rssv.rb
$ ./rssv.rb
```
